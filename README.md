# Jornada de Projetos de Pesquisa em Fisioterapia – UPE Petrolina

Site oficial da Jornada de Projetos de Pesquisa em Fisioterapia 2026 da Universidade de Pernambuco (UPE) – Campus Petrolina.

## 📖 Sobre o Projeto

A Jornada de Projetos de Pesquisa em Fisioterapia é um evento acadêmico voltado à divulgação científica e à apresentação pública dos projetos de pesquisa desenvolvidos por estudantes do curso de Fisioterapia da Universidade de Pernambuco (UPE) – Campus Petrolina.

Este website foi desenvolvido para centralizar informações do evento, incluindo programação, áreas temáticas, avaliadores, coordenação e orientações gerais aos participantes.

## ✨ Funcionalidades

- Página inicial com apresentação institucional do evento;
- Navegação intuitiva por seções;
- Exibição das datas oficiais da jornada;
- Apresentação das áreas temáticas da Fisioterapia;
- Programação completa organizada por dias e sessões;
- Sistema de abas para navegação entre os dias do evento;
- Listagem de avaliadores por sessão temática;
- Informações sobre a dinâmica e formato do evento;
- Área de coordenação;
- Design responsivo para computadores, tablets e smartphones.

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/jornada-projetos-fisioterapia-2026.git
```

2. Acesse a pasta do projeto:

